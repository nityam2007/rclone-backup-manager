# Contributing

Want to help? Great.

## Bugs
Found a bug? Open an issue. Tell me:
- What happened?
- How to reproduce it?
- Your OS and Python version.

## Features
Have an idea? Open an issue and let's discuss it.

## Pull Requests
1. Fork it.
2. Create a branch (`git checkout -b feature/cool-thing`).
3. Code it. Keep it clean (PEP 8).
4. Test it (Windows & Linux if possible).
5. Push and open a PR.

## Style
- Use type hints.
- Add docstrings.
- Keep it simple.
